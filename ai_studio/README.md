# Google AI Studio

This folder contains system instructions that were tested in AI Studio.

## Content

| Path | Description |
|---|---|
| `uber_prompt.md` | A prompt generator based on the intentions / goals described in human language. Some of other system instructs are generated using it. |
| `critical_reviewer.md` | A system prompt for reviewing content that supports constructive dialog with a user including suggestions to additional content (eg. announcements for a blog post). |
| `talk_generator.md` | A system prompt for creating talk submissions for conferences. |
